# 🎵 MusicPlayer – DevOps Web Application

## 📌 Project Description

MusicPlayer is a full-stack web application that allows users to register, log in, and manage music content.  
The project is built with a modern DevOps approach, including CI/CD pipelines, containerization, Kubernetes orchestration, and Infrastructure as Code.

The goal of this project is to demonstrate a complete production-like workflow from development to deployment.

---

## 🚀 Features

- User authentication (login / register)
- Music upload and playback system
- MySQL database integration
- REST API backend
- React frontend
- Containerized microservices (Docker)
- Kubernetes deployment
- GitOps deployment using ArgoCD
- CI/CD automation with GitHub Actions
- Pre-commit hooks (Husky) for code quality & security checks
